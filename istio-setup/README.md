# Istio Setup

This folder contains the scripts and configurations to install Istio and its observability add-ons (Kiali, Jaeger, Prometheus, and Grafana).

## Prerequisites
- Kubernetes cluster (e.g., Minikube)
- `istioctl` installed (or run `install-istioctl.sh` to install it locally)

## Steps

### 1. Install Istio CLI
```bash
./install-istioctl.sh
```
